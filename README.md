# 🏦 Banco Digital em Python

Sistema bancário simples desenvolvido em Python, simulando as principais operações de uma conta digital: depósito, saque, transferência e consulta de extrato.

## 💡 Sobre o projeto

Este é um projeto pessoal criado para praticar lógica de programação, estruturação de funções e manipulação de estruturas de dados em Python. A ideia foi simular o funcionamento básico de um banco digital, aplicando conceitos fundamentais de desenvolvimento de software.

## ⚙️ Funcionalidades

- 💰 Depósito
- 💸 Saque
- 🔄 Transferência entre contas
- 📄 Consulta de extrato

## 🛠️ Tecnologias utilizadas

- Python
